YAML 轉 JSON 工具

把 7-Eleven 門市 YAML 貼到左邊，按下「轉換成 JSON」後，右邊會輸出格式化好的 JSON。
支援目前這份資料結構：StoreId: { store: 門市名稱, address: 地址 }

就緒
輸入 YAML
把原始內容貼在這裡
輸出 JSON
會自動排版成漂亮格式
原始 YAML 來源： https://raw.githubusercontent.com/Cojad/taiwan-7Eleven-store/refs/heads/master/stores.yaml
轉換規則：頂層 key 會轉成 StoreId，裡面的 store 轉成 StoreName，address 轉成 Address。
